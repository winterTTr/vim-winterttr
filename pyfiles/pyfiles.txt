1.  Introduction                                               *pyfiles-intro*

As we know, Vim is one of the best editor. And, python is one of the most
powerful script language. And how about programming functions for vim via
python script? You, may use all the rich packages of python standard library
to accomplish the functions you want. And this thought makes the |pyfiles|
come out.

The |pyfiles| is a python package which just could be used in the vim
enviroment. This package wraps the basic elements and functions of the vim,
and you could control them via the python script. 

And before using the |pyfiles| to make the script your self, I strongly
recommend you to read the documentation about how to use the python in vim
by typing >
 :help python
